This function is for compatability with existing roms and tools. It accepts a pointer with size of 1024 which includes the attribute table. Offsets can be used for more control. ◆ unes_set_scanline_interrupt_counter()

void unes_set_scanline_interrupt_counter ( uint8_t  counter)

Sets the scanline interrupt counter.

It counts down at the end of rendering one scanline. If it hits zero, the interrupt is called and the next scanline is affected. For example, if the counter is set to 15, then the interrupt is called before rendering the 16th scanline. The 16th scanline and beyond can be modified. The counter can be set again to interrupt again later in the same frame.

Parameters
counter
